# Security Policy

## Supported Versions

Currently, we provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| v0.4.x  | :white_check_mark: |
| < v0.4  | :x:                |

## Reporting a Vulnerability

We take the security of the Agent Dashboard seriously. If you believe you have found a security vulnerability, please do **not** report it via a public GitHub issue. Instead, please follow this process:

1.  **Private Disclosure**: Email your findings to **[agent-dashboard-security@redshade.net](mailto:agent-dashboard-security@redshade.net)**.
2.  **Acknowledgement**: You should receive an acknowledgement of your report within 48 hours.
3.  **Validation**: We will investigate the report and determine if it is a valid vulnerability.
4.  **Resolution**: If valid, we will work on a fix. We may contact you for further information or to test the proposed fix.
5.  **Public Disclosure**: Once a fix is released, we will coordinate a public disclosure. We ask that you do not disclose the vulnerability publicly until a patch is available and has been announced.

Thank you for helping keep the Agent Dashboard secure!
